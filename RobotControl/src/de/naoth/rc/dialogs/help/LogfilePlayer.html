LogfilePlayer

This player only works if the LogSimulator JNI is properly setup. This includes compiling the LogSimulatorJNI project and copying the created logsimulator.dll and the Config folder to the RobotControlGUI folder.

  • reset: resets the current frame to the minimum frame number
  • run: runs the logfile in a loop
  • goto: This only works if this exact frame can be found
