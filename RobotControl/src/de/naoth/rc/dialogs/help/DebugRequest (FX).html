DebugRequest (FX)

This dialog shows a list of available debug requests for the currently connected nao robot. Note that only DebugRequests are shown that reside in currently active Modules. The hierarchy of the DebugRequests is defined by the prefixes of their names. The behavior of a DebugRequest is defined inside the corresponding module.

  • Update: Updates the list of available DebugRequests. New modules that were activated after the last update are appended to the list.

Shortcuts

  • Alt+Shift+U: Updates the list of DebugRequests (see Update).
  • F5: Updates the list of DebugRequests (see Update).
  • Space/Enter: Enables/Disables a selected DebugRequest.
  • Arrow keys: Navigate through the list and un-/collapse the tree nodes.
