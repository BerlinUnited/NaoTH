Parameter (FX)

View and change parameters.

  • Update: Updates the list of parameter files
  • Export: Save individual configurations
  • Export All: Saves all configuration files
  • Scheme: Shows the current active scheme; if no scheme is used, default is shown
  • Quicksearch: With focus on the parameter list, a filter search can be activated by simply typing characters to search for. A textfield is shown on the bottom right with the typed characters and the list/tree is filtered for elements containing the searched phrase (case insensitive).
Note: Save All for logfiles run with Logsimulator JNI only works while the logfile is run.

Shortcuts

  • Alt+Shift+U: Updates the list of parameters
  • Alt+Shift+E: Save individual configurations
  • Alt+Shift+A: Save all configurations
  • F5: Updates the list of parameters
  • Ctrl+S / Enter: in the paramter editor saves the current parameter configuration on the robot
  • Single click on a parameter: Shows the parameter values in the editor panel
  • Double click on a parameter: Shows the parameter values in the editor panel and moves the focus to the editor
  • Escape: Clears & hides the quicksearch and restores the filtered tree/list
