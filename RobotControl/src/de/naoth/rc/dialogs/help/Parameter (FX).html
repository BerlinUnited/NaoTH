Parameter (FX)

View and change parameters.

  • Update: Updates the list of parameter files
  • Export: Save individual configurations
  • Export All: Saves all configuration files
  • Scheme: Shows the current active scheme; if no scheme is used, default is shown
Note: Save All for logfiles run with Logsimulator JNI only works while the logfile is run.

Shortcuts

  • F5: Updates the list of parameter files
  • Ctrl+E: Save individual configurations
  • Ctrl+S / Enter: in the paramter editor saves the current parameter configuration on the robot
