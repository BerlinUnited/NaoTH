ModuleConfiguration (FX)

The upper right panel shows all the modules and their active/inactive status.
The left panel shows the required and provided representations for one selected module or it shows the providing and requiring modules of a selected representation.
The lower right panel shows potential problems with the current module configuration.

  • Update: Get list of modules
  • Save on robot: Copy updated modules.cfg to the robot
  • Export locally: Save modules.cfg to the selected folder
  • Quicksearch: With focus on the module tree, a filter search can be activated by simply typing characters to search for. A textfield is shown on the bottom right with the typed characters and the list/tree is filtered for elements containing the searched phrase (case insensitive).

Shortcuts

  • Alt+Shift+U: Updates the module list
  • Alt+Shift+S: Saves the module configuration on the robot
  • Alt+Shift+E: Exports the module configuration to the selected file
  • F5: Updates the module list, if the module tree has the focus
  • Escape: Clears & hides the quicksearch and restores the filtered tree/list
