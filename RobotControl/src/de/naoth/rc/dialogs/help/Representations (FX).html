Representations (FX)

This dialog shows the representations. The data can be received from the real or simulated robot or read from a logfile. Note that the contents of each representation is not always shown here. It depends on the print method of each representation.

  • Refresh: Updates the list of representations
  • Binary: Shows the deserialized protobuf message
  • Log: Shows the deserialized protobuf message (representations) of the LogFileInspector
  • Cognition/Motion: Switch between Cognition and Motion blackboard

Shortcuts

  • Alt+Shift+R: Updates the list of representations and set print mode
  • Alt+Shift+B: Updates the list of representations and set binary mode
  • Alt+Shift+L: Updates the list of representations based on the log file
  • F5: Updates the list of representations based on the currently active mode (print, binary)
  • Ctrl+F: Shows a search bar in order to find a value of the selected representation group (case sensitive!)
    • F3/Enter: marks the next search result
    • ESC: hides the search bar
  • F3: Shows the search bar or continue the search
