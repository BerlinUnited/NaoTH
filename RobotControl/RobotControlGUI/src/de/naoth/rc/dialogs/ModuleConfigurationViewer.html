ModuleConfigurationViewer

The upper right panel shows all the modules and their active/inactive status.
The left panel shows the required and provided modules for one selected module in the upper right panel
The lower right panel shows potential problems with the current module configuration.

  • Refresh: Get list of modules
  • Save locally: Save modules.cfg to private folder
  • Save on robot: Copy updated modules.cfg to the robot
  • Cognition/Motion: Switch between Cognition and Motion modules.
