RobotHealth

This dialog is used to monitor the status of critical components of the robot.

  • Circle for each joint: the color indicates the temperature, i.e., green = low, red = high;
  • Plots for each joint: illustrate the electric current consumed by the particular joint;
  • circles of the feetthe color indicates the force applied to a particular FSR, i.e., the more red, the higher the force, BLACK stands for a value which is unusually high (often it indicates a broken sensor);
