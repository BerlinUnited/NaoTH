ParameterPanel

View and change parameters.

  • update: Updates the list of parameter files.
  • Get: get parameters from robot/logfile
  • Set: set changed parameters. The new parameters are written to the privat folder
  • Save: Save individual configurations
  • Save All: Saves all configuration files.
Note: Save All for logfiles run with Logsimulator JNI only works while the logfile is run.
