Plot2D

Produces a 2D graph of a integer or floating point value over time. Usually a DebugRequest exists for each plot that must be enabled before it can be selected for plotting.

  • Receive Data: Start receiving the data.
  • Clear: Clears the list of available plot, the graph and received data.
  • Clear Data: Clears the graph and received data.
  • Fit to Data: Rescales the graph such that all received data is visible in the graph
  • export csv: Saves the already received data in a txt file located at in the RobotControlGUI folder.
  • export graph: Saves the graph as image. The destination and file format can be selected.
