RepresentationInspector

This dialog shows the representations. The data can be received from the real or simulated robot or read from a logfile. Note that the contents of each representation is not always shown here. It depends on the print method of each representation.

  • Refresh: Updates the list of representations.
  • Binary: Shows the deserialized protobuf message
  • Log: TODO
  • Cognition/Motion: Switch between Cognition and Motion blackboard.
