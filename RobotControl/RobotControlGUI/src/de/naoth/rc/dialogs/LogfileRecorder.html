Logfile recorder

This dialog is used to record logfiles.

Important: Please mind that you can't use unlimited space on the robot. When logging images you normally are not able to log more than one minute.

Even more important: The logfile will be saved on the robot to the tmp directory by default. Thus, it will be deleted when you restart or shutdown the robot. Therefore, be sure to have copied the recorded logfile to you local storage before shutdown the robot :)

To record a logfile follow these steps:

1. Choose wether you want to make a logfile from the Cognition or Motion process.
2. Check the path to the file on the robot that should be used as temporay storage for the logfile.
3. Click on the "New" icon.
4. Choose the representations to log.
5. Click the record button when you want to start your log. You can always suspend logging by clicking on it and resume later. It is even possible to change the logged representations when not logging.
6. To finish recording click the "stop" button.
7. Click the "save" button to download the recorded file from the robot. You can also get the file directly from the robot (e.g., using scp) from the Temporary logfile path.
