BehaviorViewer

This dialog shows the currently executed behavior of the connected Nao or LogSimulator.

  • Receive: Receives the behavior from the real or simulated robot or from an executed logfile.
  • Receive Log: This button is only used together with the LogfileInspector dialog
  • Only Options: Only shows the options.
  • Send to Robot: Send a new compiled behavior to the robot. This can be done at any time while the robot is doing something else.

With the Add Watch Button the values of individuall symbols can be tracked during the behavior execution.
