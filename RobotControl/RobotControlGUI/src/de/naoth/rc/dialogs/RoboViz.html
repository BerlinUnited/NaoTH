RoboViz

Allows the integration of the RoboViz application into RobotControl.

Once the RoboViz jar file was selected or dropped onto this panel, the path to the jar file is stored in the configuration file of RobotControl. To (re-)select or modify the path of the RoboViz jar file, edit the RobotControl configuration file or just move/rename the old. The RobotControl configuration can be found in your users home directory under: "~/.naoth/robotcontrol/config".

The RoboViz configuration is stored in a seperate config file under: "~/.roboviz/config.txt"!
