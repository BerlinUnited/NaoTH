LogfilePlayer

A graphic user interface for logsimulator. It uses java to call logsimulatior and use pipe stream to control it.

The usage is very similar as movie players: use open to open a logfile, then you can play or forward/backward step by step, or jump to the given frame, and stop to quit the logsimulator.

You can also use keyboard control the playing as well, the defination of keyboard is the same as logsimulator.
